<h2>Boss Project</h2>

This was a project for my Artificial Intelligence Programming class. All the assets were obtained from the Unity Asset store and Mixamo.
In this project, I was tasked to desinged a boss fight.  I had to code the bosses' behavior (Mr. Bateye's behavior) utilizing the state machines. 
I also took artistic liberties and added cut scenes using Cinemachine and splines.
